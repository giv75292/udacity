## 纳米学位值不值得？听听学过的人怎么说

网上找了一圈，Udacity的口碑还真不错。Trustpilot上有3000多条评价，总分4.6/5，80%以上给了5星好评。

**学员普遍认可的点：**

1. **课程质量扎实**：视频讲解清晰，每个视频3-5分钟，知识点拆解得很细。不会让你看半小时还不知道在讲啥。

2. **项目导向学习**：每个纳米学位都有实战项目。不是那种"跟着敲代码"的项目，而是开放式的，需要你自己思考解决方案。完成后有专业导师审阅，给反馈。

3. **导师支持给力**：遇到问题可以在论坛提问，通常几小时内就有回复。有些课程还配1对1导师辅导。

4. **职业服务实用**：完成纳米学位后，平台提供简历优化、LinkedIn主页优化、GitHub代码审查等服务。有些课程还有模拟面试。

**也有人吐槽的地方：**

- 部分课程内容更新不够及时（虽然Udacity一直在更新，但技术发展太快）
- 客服响应有时候慢
- 没有中文字幕的课程对英语不好的人不太友好（不过主流课程基本都有中文字幕）

一个学过7个纳米学位的老哥总结得挺到位："Udacity教的东西确实能让你入门，但想真正精通，光靠上课不够，得自己大量练习。"

这话说得实在。在线教育都这样，关键看你自己愿意投入多少。

## 和其他平台比，Udacity的优势在哪？

市面上在线学编程的平台一大把，凭什么选Udacity？

**Coursera：理论派，学术气息重**
大学教授讲课，偏理论。适合想系统学习基础知识的人。价格便宜，但实战项目没Udacity那么深入。

**Udemy：课程海量，质量参差**
什么课都有，从$10到$200不等。但课程质量全靠讲师个人，没统一标准。

**Codecademy/FreeCodeCamp：适合纯小白入门**
交互式学习体验不错，但深度有限。学完之后很难直接找到工作。

**Udacity：实战导向，对标就业**
课程都是跟业界合作开发的，教的就是公司需要的技能。项目审阅、职业服务都更完善。价格贵一些，但如果真能因此找到工作或升职加薪，这钱花得值。
